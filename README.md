# Hackerboost-JavaScript-Warm-up
This project is a hands-on practice of core JavaScript concepts, focusing on how JavaScript interacts with HTML, how strict mode improves code safety, how functions handle parameters and defaults, and how conditional logic can be used to build real logic like a grade calculator.

---

## What the Project Does

This project contains four mini-tasks:

### 1️⃣ Smart Script Placement
- HTML page with a button that updates text when clicked
- Uses **external JavaScript**
- Demonstrates best practice script placement for performance

### 2️⃣ Strict Mode Detective
- Shows the difference between JavaScript with and without `"use strict"`
- Demonstrates how strict mode prevents accidental global variables

### 3️⃣ Personalized Welcome System
- Function that accepts a name and membership status
- Uses default parameters
- Uses conditionals to display different messages

### 4️⃣ Grade Calculator with Feedback
- Function that takes a score (0–100)
- Returns a grade and a feedback comment
- Handles edge cases and invalid inputs

---

## Technologies Used

- HTML5
- Vanilla JavaScript (ES6)
- DOM Manipulation
- Node.js (for running JavaScript in the terminal)

## ▶️ How to Run the Project

### Option 1 — Run in the Browser

2. Right-click `index.html`
3. Select **Open with Live Server**
4. Click the button on the page to see JavaScript in action

### Option 2 — Run in the Terminal (Node.js)
